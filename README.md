# Chaos Pizza API

Simple pizza delivery API.

## Install
npm install

## Run
npm start

## Tests

### Exécuter les tests
```bash
npm test
```

### Exécuter les tests avec couverture de code
```bash
npm run test:coverage
```

Le rapport de couverture sera généré dans le dossier `coverage/` avec :
- **Rapport texte** dans la console
- **Rapport HTML** dans `coverage/lcov-report/index.html` (ouvrez-le dans votre navigateur)
- **Rapport LCOV** pour l'intégration avec d'autres outils

### Seuils de couverture configurés
- Branches : 50%
- Fonctions : 50%
- Lignes : 50%
- Statements : 50%
